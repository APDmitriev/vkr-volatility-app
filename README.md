# Приложение для анализа и прогнозирования временных рядов

ВКР на тему: **«Разработка приложения для анализа и прогнозирования значений волатильных временных рядов»**.

## Описание

Приложение предназначено для загрузки, анализа, предобработки и прогнозирования временных рядов.

Система позволяет работать с данными через графический интерфейс, запускать разные модели прогнозирования, сравнивать результаты и формировать отчёты.

## Возможности

- импорт данных из CSV и XLSX;
- предобработка временных рядов;
- анализ волатильности;
- построение графиков;
- прогнозирование значений;
- сохранение экспериментов;
- сравнение моделей по метрикам;
- экспорт отчётов.

## Модели

В приложении используются следующие модели:

- Linear Regression;
- ARIMA;
- SARIMA;
- GARCH;
- Fuzzy Time Series;
- MLPRegressor;
- LSTM.

## Запуск backend

```powershell
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Backend будет доступен по адресу:

```text
http://127.0.0.1:8000
```

## Запуск интерфейса

```powershell
cd vkr_pyqt_ui
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

## Пример данных

Для проверки можно использовать CSV-файл с колонками:

```text
date,open,high,low,close,volume
```

Для прогнозирования можно выбрать целевую колонку `close`.
